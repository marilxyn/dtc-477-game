About the Project

Project Description

Lewis & Clark: The Great Expedition is an educational historical adventure game that follows the journey of Meriwether Lewis and William Clark as they explore the newly acquired western territories of the United States.

GitHub Collaboration

This project was built collaboratively using GitHub. Each developer was responsible for specific chapters and assets throughout the development process. Folders were organized to store shared resources such as images and audio files. Each push to the repository contained the latest updates and contributions from all team members. Using GitHub provided an effective way to manage code development and track progress over time.
